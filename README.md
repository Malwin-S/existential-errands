# 🦉 Existential Errands

**Mundane tasks for a meaningless universe.** Existential Errands is a "Progressive Web App" (PWA) designed to gamify your daily survival. Complete real-world tasks, earn XP, and interact with Hoot—a slightly depressed owl narrator who doesn't care about your productivity, but tracks it anyway.

## ✨ Features
- **Zero Cost:** No subscriptions, no ads, no "penny-pinching" mechanics.
- **Privacy-First:** 100% of data is stored in your local browser. We know nothing.
- **Installable:** Works as a native app on Android and iOS (PWA technology).
- **The Streak of Shame:** Track your daily consistency (or lack thereof).

## 🛠️ Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Progressive Web App (PWA) with Service Workers
- LocalStorage for persistent data
- Hosted for free on GitHub Pages

## 🚀 How to Install
1. Visit the [Live Link](https://malwin-s.github.io/existential-errands/)
2. **On Android:** Tap "Add to Home Screen" when prompted.
3. **On iOS:** Tap the 'Share' icon and select "Add to Home Screen."
